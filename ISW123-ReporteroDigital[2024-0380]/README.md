
Aplicación de consola en C#.

## Descripción
Este programa simula un reportero digital que arma una noticia usando:
- Texto
- Imagen
- Análisis

La información se obtiene en paralelo para que el sistema sea más rápido.  
El programa también maneja errores de red sin cerrarse.

## Conceptos aplicados
- Programación asíncrona (`async` y `await`)
- Ejecución paralela de tareas (`Task` y `Task.WhenAll`)
- Manejo de eventos (`event` y `Action`)
- Manejo de excepciones (`try-catch`)
